This mod disables quickswapping items inside inventories.

This includes moving items back and forth from the hotbar using the 1-9 keys, and sending items to the offhand slot by pressing F inside an inventory.

This mod prevents Dupe Exploits involving unhandled quickswapping in item containers[eg backpacks] in a number of mods including but not limited to:
Immersive Engineering
Applied Energistics 2
Engineer's Tools
Corail Backpack
Tetra
Blood Magic
